# Screenshots Directory

This directory contains screenshots of the application from a user's perspective.

## Required Screenshots

Please add the following screenshots to this directory:

1. **home-page.png** - The home/welcome page
2. **todo-list.png** - The todo list view with todos displayed
3. **create-todo.png** - The form for creating a new todo
4. **edit-todo.png** - The form for editing an existing todo
5. **search-results.png** - Search results with filters applied
6. **404-error.png** - Custom 404 Not Found error page
7. **500-error.png** - Custom 500 Internal Server Error page
8. **swagger-ui.png** - Swagger UI showing API documentation
9. **api-response.png** - Example JSON response from the REST API

## How to Take Screenshots

1. Start the application: `mvn spring-boot:run`
2. Navigate to http://localhost:8080
3. Take screenshots of each page/feature
4. Save them in this directory with the names listed above
5. Update the README.md file paths if you use different names

## Screenshot Guidelines

- Use PNG format for best quality
- Recommended resolution: 1920x1080 or higher
- Include the full browser window (address bar visible)
- Show actual data/functionality, not empty states
- Ensure text is readable and UI elements are visible
